Anotacija

Šiame projekte kuriama intelektuali duomenų analizės sistema, skirta automatizuotai atpažinti pasikartojančius elgsenos modelius finansinių duomenų srautuose.

English: This project develops an intelligent data analysis system designed to automatically detect recurring behavioral patterns in financial data streams.

Raktažodžiai: duomenų analizė, dirbtinis intelektas, mašininis mokymasis

  • II semestras – tarpinis atsiskaitymas, prototipas
  • III semestras – galutinė dokumentacija, gynimo medžiaga
